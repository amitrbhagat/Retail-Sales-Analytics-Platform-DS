# Retail Sales Analytics & Forecasting Platform

## Overview

An end-to-end full-stack AI and Data Science platform for retail analytics, sales forecasting, inventory recommendations, and AI-powered business insights using a local LLM with Ollama.

## Tech Stack

- Frontend: React
- Backend: Django REST Framework
- Database: PostgreSQL
- Machine Learning: Scikit-learn, XGBoost, LightGBM, Prophet
- AI Assistant: Ollama
- Visualization: Chart.js / Plotly
- Containerization: Docker
